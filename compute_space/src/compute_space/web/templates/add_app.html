Deploy from Git URL

Clone a git repository containing an openhost.toml manifest file. Private GitHub repos are supported automatically — you'll be prompted to connect your GitHub account if needed. You can also access private repos by including an access token directly in the URL.

To specify a branch, tag, or commit, append @ref to the URL:

  • https://github.com/user/repo — default branch
  • https://github.com/user/repo@my-branch — specific branch
  • https://github.com/user/repo@v1.0 — tag
  • https://github.com/user/repo@abc123 — commit hash
  • https://<token>@github.com/user/private-repo — private repo via access token
